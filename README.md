## :bangbang: Folder Structure

Here is the folder structure of this app.

```bash
3d-portfolio/
  |- public/
    |-- fonts/
    |-- images/
    |-- apple-touch-icon.png
    |-- style.css
    |-- me2.jpg
  |- src/
    |-- animation/
        |--- contact.json
        |--- dev.json
        |--- done.json
    |-- components/
        |--- 1-header/
        |--- 2-hero/
        |--- 3-main/
        |--- 4-contact/
        |--- 5-footer/
    |-- app.tsx
    |-- index.css
    |-- main.tsx
  |- .eslintrc
  |- .gitignore
  |- index.html
  |- package-lock.json
  |- package.json
  |- jsconfig.json
  |- vite.config.ts
```
